# Javascript_Racer
A simple two player Javascript Racer

### ONLY WORKS IN CHROME AT THE MOMENT

## Run locally
Install all the dependencies:

  `bundle install`
  
Then run shotgun in your command line:

  `bundle exec shotgun`
